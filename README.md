# Hen Project

Simple PySide6 starter project for Windows desktop development.

## Settings

- Global app settings are centralized in `src/settings.py`
- Change app name, language direction, and window size from one place

## Run

1. Activate virtual environment:
   - PowerShell: `.\venv\Scripts\Activate.ps1`
2. Install requirements:
   - `pip install -r requirements.txt`
3. Start app:
   - `python src/main.py`
